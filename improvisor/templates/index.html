Improvisor is an application that will allow you create a presentation dynamically from previously saved assets and associated tags. Improvisor uses speech recognition to look for tags in the presenter speech and presents the linked assets on a mobile device for the presenter to choose and display. At the end of a presentation you can export shown assets into a slide set which can be shared with the audience.
