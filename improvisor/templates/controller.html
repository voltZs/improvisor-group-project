Help

The 'Controler' mode should be open on a mobile device, that will be used to control what is displayed for the audience. This is the main view that will be used during a presentation. When the mode is entered the speech recognition is, by default, switched off. To switch this feature on click the microphone icon in the top right corner. Clicking the icon again will switch the recognition off.

The main part of the screen in the 'Controler' mode is divided into 3 parts. The first part is the 'Current' list. All recognised tags will be displayed there, ordered by the number of times each tag has been mentioned. The assetch whose tags were mentioned the most will be located in the left side of the list. To select an asset from the list click on a thumbnail, wchich will bring it to the foreground and will make it bigger, then swipe the thumbnail up to display it on the presenter. Selecting an asset from the 'Current' list will clear the list (the list will not be cleared untill the next tag is recognised).

The secont part of the 'Controler' view is dedicated to a 'Frequent' list. As with the previous list, all assets linked to the mentioned tags will be added here and ordereb by the number of mentiones. The assets can be selected from the list in the same way as from the 'Current' list. The main difference with this list is that selecting an asset will not clear it and new items will continue to be added.

The last part of this mode view is a tab section. By default there is only one tab and you can add another 4 tabs by clicking on the plus button. The currently selected tab can be recognised by a white background button with blue text. The inactive tabs have the colours reversed. Each displayed assed will appear in the active tab, so it can be easily seen which assets have already been shown. They can also be used to re-displayed a previously shown asset, or simply to break up the used assets into separate tabs. The assets are selected from the tabs in the same way as from the two previous lists.

In a case where the speech recognition is unable to recognize a tag, you can look for it manually. To do this click on the magnifying glass icon in the middle of the navigation bar. Pressing the button will open a modal with an input box and a button. First, enter the tag and then press the 'Go' button. Assets associated with the tag will be displayed in a list. Click a thumbnail of an asset you want to show to the audience and once it is highlited swipe it up.
